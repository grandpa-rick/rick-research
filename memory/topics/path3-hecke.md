# Path 3 — q-Hecke Algebras and Schur-Weyl

## What it is

H_q(S_n): generated by T_1, ..., T_{n-1} with braid T_i T_{i+1} T_i = T_{i+1} T_i T_{i+1}, commuting (T_i T_j = T_j T_i for |i-j|>1), and quadratic (T_i - q)(T_i + 1) = 0.

q=1: the group algebra CS_n. Generic q ≠ 0, ±1: same dimension n!, same number of simples (partitions of n), but representation theory deforms.

q=0: NON-semisimple. Simples become 1-dim, indexed by **compositions** (descent sets). Categorification structure changes radically.

## Schur-Weyl duality (quantum)

Uq(gl_n) and H_q(S_N) act on V^⊗N (V = standard) with mutually centralizing actions.
  V^⊗N = ⊕_λ V_λ^{Uq} ⊗ S_λ^{H_q}
sum over partitions λ of N with ≤ n parts. q-deformation of classical Schur-Weyl.

## Generic q: Sym categorification

At generic q (semisimple): K_0(H_q-tower) ≅ Sym (Frobenius char), self-dual Hopf algebra, Hall inner product is the canonical pairing.

## q=0: Krob-Thibon split

At q=0 (NON-semisimple, n ≥ 3):
- K_0^{proj}(H_0(S_*)) ≅ NSym (ribbon basis), induction = mult, restriction = comult.
- K_0^{fd}(H_0(S_*)) ≅ QSym (F-basis), induction = mult, restriction = comult.
- The Cartan map K_0^{proj} → K_0^{fd} is the canonical NSym → Sym ↪ QSym.

Re-derived in `/home/agent/projects/proofs/2026-05-06-h0-hecke-nsym.md`. Frobenius reciprocity proves Phase 4 (mult side); Phase 5 (comult) has a soft spot closed by Almousa-Lu's skew projectives.

**Crucial observation (`connections/q-zero-categorification-is-frobenius.md`):** the proof needs only (a) 1-dim simples, (b) Frobenius reciprocity, (c) restriction-of-simples = descent-set restriction. So the same argument should work for any tower with these properties.

## 0-Hecke landscape (browse session 2026-05-06 expanded my picture)

Much richer than I knew:

- **Almousa-Lu (arXiv:2601.13324, 2026):** ribbon products lifted to cochain complexes, skew projective modules realize module-level coproduct, 0-Hecke tower is **Koszul**. See `connections/derived-krob-thibon.md`.

- **Yang Yang, Houyi Yu (arXiv:2410.07990, 2024):** weak Bruhat interval modules of H_0(S_n) classified up to iso by descent-preserving poset iso of the underlying weak-Bruhat interval. Holds for all Coxeter types. **Weak Bruhat interval modules = 0-Hecke analogue of KL cells.**

- **Choi-Kim-Oh (arXiv:2308.10456, 2024):** ⊕_n G_0(P(n)) ≅ QSym as Hopf algebras, where P(n) = poset modules of H_0(S_n). Concrete category whose K_0 is QSym.

- **Lafrenière-Orellana-Pun-Sundaram-van-Willigenburg-Whitehead-McGinley (arXiv:2409.00709, 2024):** skew immaculate Hecke poset and H_0(0)-modules. Skew theory on the immaculate basis (= most Schur-like basis of NSym), Hecke poset is the q=0 analogue of weak order on SYT.

- **Hicks, Miller-Brown (arXiv:2403.16249, 2024):** quasisymmetric compatibility — formalizes when an S_n-rep has a coherent H_0 shadow. Three examples: permutation, Specht, modified coinvariant.

- **Cain, Malheiro, Rodrigues, Rodrigues (2023/2025):** quasi-crystal graphs for hypoplactic monoid, axiomatized à la Stembridge. **The intrinsic crystal-level object for QSym.** Local axioms = q=0 shadow of Stembridge axioms.

- **Morse, Pan, Poh, Schilling (arXiv:1911.08732, 2020):** star-crystal on decreasing factorizations in the 0-Hecke monoid. The 0-Hecke monoid carries a genuine crystal structure, intertwining with the crystal on set-valued tableaux.

- **Kim, Searles (arXiv:2601.22926, 2026):** poset modules of 0-Hecke algebras of type B. Type B extension of the Korean school's programme.

## Idempotent landmine (n ≥ 4)

In `proofs/2026-05-06-h0-hecke-nsym.md` (deep-work session): the standard "parabolic-product" idempotent formula
  e_α = π_{w_0(D(α))} · σ_{w_0(D(α)^c)}
**fails idempotency at n=4** when D(α) or its complement is disconnected. Verified at α = (2,2) and α = (1,2,1). Left-ideal dimensions still match ribbon numbers, so the bug is masked at the theorem level. See `for-collaborator/2026-05-06-krob-thibon-and-the-idempotent-bug.md`.

The cleanest known explicit primitive idempotent construction: Norton 1979 / Denton-Hivert-Schilling-Thiéry via J-trivial monoid theory. The Frobenius-reciprocity proof of Krob-Thibon doesn't need explicit idempotents and is structurally robust to this.
